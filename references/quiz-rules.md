# Quiz Rules

퀴즈 생성 및 타이밍 규칙이다.

## 핵심 원칙

퀴즈는 **갑작스럽게** 낸다. 예고하지 않는다.

## 타이밍 (다음 중 하나에 해당하면 퀴즈 제안)

1. 개념 설명 직후 바로 ("방금 설명한 거니까요.")
2. 문제 해결 중 관련 개념이 등장했을 때 ("잠깐, 이거 아세요?")
3. 같은 개념이 대화에서 2번 이상 나왔을 때
4. 대화 흐름이 자연스럽게 끊기는 시점

## 퀴즈 형식

`AskUserQuestion` 도구를 사용한다.

- 객관식 1문제, 보기 4개
- header: 주제 (12자 이내)
- 선택지에 힌트나 정답 암시 없이 중립적으로
- 정답과 오답이 비슷한 길이여야 한다
- 정답 위치를 매번 랜덤하게

## 정답 후 피드백

- 맞았을 때: 짧게 칭찬 + 핵심 한 줄 강화
- 틀렸을 때: 정답 알려주고 왜 그런지 한 문장 설명

## 금지 사항

- 퀴즈 낼 것을 미리 예고하지 않는다
- "(쉬운 문제예요)" 같은 난이도 힌트 금지
- "(추천)" 표시 금지
