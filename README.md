# challenge-guide 🧭

  > 답을 알려주지 않는 AI 학습 가이드

  ## 소개 💡

  질문에 바로 답하지 않습니다.
  비유와 힌트로 스스로 발견하도록 안내하고, 공식문서를 직접 탐색하게 유도합니다.
  그리고 대화 중 예고 없이 퀴즈를 냅니다.

  ## 기능 ✨

  ### 🔍 개념 설명
  일상적인 비유 → 한 줄 정의 → 최소 예시 코드 순서로 설명합니다.
  전문 용어 대신 누구나 아는 소재로 감을 잡게 합니다.

  ### 💡 문제 해결 힌트 (3단계)
  1. **영역 좁히기** — 어느 방향을 봐야 하는지만 알려줌
  2. **구체적 단서** — 조금 더 구체적인 실마리 제공
  3. **거의 정답** — 한 발짝만 더 생각하면 풀 수 있는 힌트

  3단계 후에도 해결 못하면 그때 직접 설명합니다.

  ### ⚡ 퀴즈
  대화 흐름 중 갑자기 등장합니다. 예고 없습니다. 주의하세요.

  ### 📝 세션 저장
  대화가 끝나면 Notion에 오늘 학습 내용을 정리해 저장합니다.

  ## 사용법 🚀

  Claude Code에서 `/challenge-guide`로 실행합니다.

  | 입력 유형 | 예시 | 동작 |
  |---|---|---|
  | 개념 설명 요청 | "옵셔널이 뭐야?" | 비유 + 예시로 설명 |
  | 문제 해결 요청 | "이 코드 왜 안 되지?" | 단계별 힌트 제공 |
  | 세션 저장 | "정리해줘" | Notion에 저장 |
